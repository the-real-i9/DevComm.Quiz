Q1: What is the value of result?

def add(a, b):
    result = a + b;
    return result;
add(5, 6);

Answer: 11

Correct Answer: 11

Explanation: The function parameters which serves as local variables are assigned its arguments in order a = 5, b = 6 the sum of a and b is computed and its result is assigned to the result variable, the return statement returns the value of the expression on its RHS back to the caller.
